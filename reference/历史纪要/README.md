# 历史会议纪要

本目录用于存储完整的会议纪要文件，作为 Few-Shot Learning 的范文。

## 文件命名规范

- 文件名格式：`管理周会纪要_YYYY-MM-DD.md`
- 示例：`管理周会纪要_2026-03-02.md`

## 使用方式

1. 上传新的会议纪要文件到 `reference/历史纪要/` 目录
2. 在 Streamlit 侧边栏"🧠 大脑控制台"的"历史纪要范文选择"部分选择
3. 最多选择 2 篇范文作为生成参考

## 注意事项

- 建议上传质量较高的会议纪要作为范文
- 定期清理过旧的范文，保持最新的 10 篇左右
- 文件内容应保持完整性，不要修改

## 自动整理

系统会自动按文件名排序，最新的文件优先显示。
